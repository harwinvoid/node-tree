# node-tree
node实现linux下Tree -L命令，展示目录。

# how to start
npm install

# how to use
### 1 展示所有目录
### tree
![](./img/tree2.png)
### node ../node-tree/index.js
![](./img/ntree2.png)

### 2 展示指定目录级数
### tree -L 1
![](./img/tree1.png)
### node node-tree/index.js -L 1
![](./img/ntree1.png)

### 3 【拓展】展示指定目录
